### Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/nghuiling/Qgen.git
   ```
3. Install Python packages
   ```sh
   pip install -r requirements.txt
   ```
4. Run Streamlit 
   ```sh
   streamlit run home.py
   ```
